A vulnerability, designated as CVE-2026-3102, was discovered in ExifTool, a utility used for reading and writing metadata in various file types. This flaw affects macOS systems with ExifTool version 13.49 and earlier, allowing an attacker to execute arbitrary shell commands with the privileges of the user invoking ExifTool. The vulnerability is triggered when the -n flag is used, which outputs machine-readable data without additional processing. Zero-Day Exploit Against Windows BitLocker

A zero-day exploit against Windows BitLocker has been discovered, allowing attackers to bypass the full-volume encryption protection. This exploit, named YellowKey, was published by a researcher known as Nightmare-Eclipse and can reliably bypass default Windows 11 deployments of BitLocker. BitLocker is a crucial security feature that makes disk contents inaccessible to anyone without the decryption key, which is stored in a trusted platform module (TPM). The exploit requires physical access to the computer, which limits its potential impact. However, for organizations that rely on BitLocker as a mandatory protection, this vulnerability poses a significant risk. Many organizations, including those that contract with governments, require BitLocker to be enabled on their systems. The fact that YellowKey can bypass this protection means that an attacker with physical access to a computer could potentially access sensitive data. ...
