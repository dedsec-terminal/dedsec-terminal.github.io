Here are three concise paragraphs summarizing the ThreatsDay Bulletin:

The internet has been used for malicious activities, including searches being siphoned through shady browser add-ons and AI chat links being turned into malware delivery paths. macOS attacks have been running in memory, leaving almost nothing behind, while cloud agents have been treated as open shells by attackers. Additionally, exposed edge gear, poisoned packages, cash courier scams, stealers, loaders, and phishing campaigns have been on the rise. Microsoft has announced the general availability of DNS-over-HTTPS (DoH) for Windows DNS Server, which aims to improve privacy, reduce spoofing risk, and advance Zero Trust DNS.

Several specific threats have been identified, including a cluster of 23 deceptive Chrome browser extensions that override users' default search engines and route queries through monetization middleware. A fileless macOS ClickFix attack chain has been observed, targeting victims in Asia, North America, and Oceania, and delivering an AppleScript-based infostealer. Claude chat abuse has also been used to deliver malware, with threat actors hijacking Google Ads searches to funnel victims towards malicious download pages. Furthermore, a WhatsApp booking fraud campaign has been spreading globally, impersonating hotels and resorts to trick travelers into handing over payment card details.

Other notable threats include the exploitation of a critical privilege escalation flaw in Cisco's Catalyst SD-WAN Controller and Manager, as well as the discovery of two high-severity local code-execution paths in the Cline AI coding agent VS Code extension. An HTTP/2 abuse vulnerability has also been identified, allowing attackers to amplify junk traffic by orders of magnitude. Additionally, an exposed email server has been used as a phishing hub, and a bank phishing campaign has been delivering an in-memory stealer. Amazon Web Services has announced a new AI-powered security agent called AWS Continuum to address code vulnerabilities, highlighting the ongoing cat-and-mouse game between attackers and defenders.
