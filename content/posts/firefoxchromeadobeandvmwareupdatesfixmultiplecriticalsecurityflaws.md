Mozilla has released updates for Firefox to address two critical security flaws, CVE-2026-15718 and CVE-2026-15719, which could be exploited to compromise user systems. Although exploit code for these vulnerabilities has been published, Mozilla is not aware of any attacks in the wild. The updates are available in Firefox version 152.0.6. Meanwhile, Google has also released updates for Chrome to fix 15 security flaws, including two critical use-after-free bugs in Ozone, which could allow remote attackers to exploit heap corruption via crafted HTML pages.

Adobe has published security updates for 88 vulnerabilities, including multiple critical-severity bugs in ColdFusion, Commerce, Experience Manager, and Illustrator. The updates address vulnerabilities that could lead to arbitrary code execution, privilege escalation, and other security risks. Eight of the vulnerabilities impact Adobe ColdFusion, with CVSS scores ranging from 9.0 to 9.9. The flaws have been remediated in versions ColdFusion 2025 Update 11 and ColdFusion 2023 Update 22. Additionally, Adobe has fixed critical flaws in Adobe Commerce, Magento Open Source, and Adobe Experience Manager.

VMware has also released a fix for a critical authentication bypass vulnerability in VMware Avi Load Balancer, which could allow malicious users to access the Avi Control plane. The vulnerability, CVE-2026-47865, has a CVSS score of 9.8 and was discovered and reported by Filip Waeytens of the NATO Cyber Security Centre. Although none of the vulnerabilities have been marked as actively exploited, it is essential for organizations to install the latest updates to prevent potential attacks. Threat actors are known to weaponize flaws in these products, making it crucial to stay up-to-date with the latest security patches.
