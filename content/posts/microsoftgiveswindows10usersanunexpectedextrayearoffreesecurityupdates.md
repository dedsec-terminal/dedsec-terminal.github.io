Microsoft has unexpectedly extended its Extended Security Updates (ESU) program for Windows 10 users by an additional year, until October 12, 2027. This program provides critical security updates for eligible devices after the official end of support, which occurred in October 2025. The extension allows users to continue receiving security updates while they transition to a new Windows 11 PC.

To be eligible for the ESU program, devices must run Windows 10 version 22H2 Home, Professional, Pro Education, or Workstations edition and have the latest updates installed. Users must also have an administrator Microsoft account, which becomes associated with the ESU license. There are three ways to enroll in the program: enabling Windows Backup, redeeming 1,000 Microsoft Rewards points, or purchasing a one-year ESU license for $30 plus taxes.

The extension is a welcome surprise for Windows 10 users, providing them with more time to upgrade to a new operating system or purchase a new PC. Microsoft continues to encourage users to upgrade to Windows 11 or purchase a new Copilot+ PC, but the extra year of security updates will help ensure their devices remain secure in the meantime. Users who are already enrolled in the ESU program will automatically continue to receive coverage until the new end date, with no action required.
