Microsoft has released a record-breaking Patch Tuesday update for June 2026, addressing nearly 200 security vulnerabilities across its Windows operating systems and supported software. Of these, 36 bugs have been rated as "critical" by Microsoft, and exploit code for at least three of the weaknesses is now publicly available. This heavy Patch Tuesday may become the norm, as both Microsoft engineers and the security community are increasingly using artificial intelligence tools to find bugs.

The updates include fixes for several zero-day vulnerabilities, including a denial of service vulnerability affecting Microsoft Internet Information Services (IIS) and an elevation of privilege weakness in the Windows Collaborative Translation Framework. One of the zero-days, dubbed "GreenPlasma," was disclosed by a security researcher known as Nightmare Eclipse, who has been releasing exploits for Windows flaws. Microsoft has patched several vulnerabilities disclosed by Nightmare Eclipse, but the researcher has pledged to release even more zero-day exploits in July.

The actual number of security flaws addressed by Microsoft this month is higher than the reported 200, with over 360 browser vulnerabilities also being patched. Other software makers, including Adobe and Google, have also released large update bundles this month. Users are advised to back up their data before applying operating system updates and to be cautious of potential issues with the patches. The increasing use of artificial intelligence tools to find bugs is expected to lead to more frequent and extensive Patch Tuesday updates in the future.
