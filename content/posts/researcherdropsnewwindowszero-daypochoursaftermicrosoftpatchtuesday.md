Here are 3 concise paragraphs summarizing the article:

Security researcher Chaotic Eclipse has released a new proof-of-concept (PoC) exploit called LegacyHive, which targets a Windows User Profile Service vulnerability. The exploit allows for arbitrary hive load elevation of privileges, and it is functional on all supported desktop and server versions of Windows, including those running the latest July 2026 Patch Tuesday update. The researcher has stripped down the exploit to prevent public exploitation, but noted that the original exploit did not require additional user credentials and was not limited to the "usrclass.dat" hive.

The release of the LegacyHive exploit comes amid a heated dispute between Chaotic Eclipse and Microsoft, which has been ongoing since at least April 2026. The researcher has been releasing details of multiple exploits before Microsoft had a chance to patch them, citing a breakdown in communication. This has led to several vulnerabilities being actively exploited, including three in Microsoft Defender. Microsoft has released security updates to address some of these vulnerabilities, but the company is still investigating the new LegacyHive exploit.

The LegacyHive exploit is not the only security concern for Microsoft, as the company recently shipped patches for a record 622 flaws, including two privilege escalation vulnerabilities in SharePoint Server that have been flagged as actively exploited. The US Cybersecurity and Infrastructure Security Agency (CISA) has added these vulnerabilities to its Known Exploited Vulnerabilities catalog, mandating that federal agencies apply the fixes by specific deadlines. Other security experts have noted that the Patch Tuesday process has experienced significant turbulence in 2026, with the emergence of a series of vulnerabilities disclosed in a way that brings maximum discomfort for Microsoft.
