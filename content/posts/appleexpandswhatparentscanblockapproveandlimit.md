Apple has announced new child safety features for iPhone, iPad, and Mac, which will be available with updates to iOS 27, iPadOS 27, and macOS 27 this fall. These features aim to help parents create age-appropriate digital experiences for their children by expanding parental controls over app access, web browsing, communication, and screen time. The updates are designed to be simple and intuitive, based on expert guidance from child development researchers, clinicians, and online safety experts.

The new features include tools such as Ask to Buy, which requires approval before a child can install apps, and Ask to Browse, which allows parents to approve or block websites accessed by their children. Parents can also manage who their children communicate with through Messages, FaceTime, and Phone, and require approval before their child connects with a new contact. Additionally, Communication Safety will intervene when violent or graphic content is detected in shared images and videos, and will blur nudity when detected. These features can be managed through a Child Account, which is required for children under 13 and available for those up to 18 years old.

The updates also provide more control over screen time, with features such as Time Allowances, which allow parents to set limits on the time children spend in certain app categories, and Daily Schedules, which enable parents to control which apps children can access during weekdays and weekends. A redesigned Screen Time dashboard provides an overview of children's device usage and most-used apps, and allows parents to adjust app access and time limits as needed. Apple has also launched a dedicated website for parents to help them stay informed about the latest tools and features, and has introduced additional tools such as Screen Time Passcode Notifications and User Reporting Tools to help protect children online.
