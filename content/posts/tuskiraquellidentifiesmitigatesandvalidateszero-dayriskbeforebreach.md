Tuskira has launched Quell, a zero-day defense capability that helps enterprises identify, mitigate, and validate zero-day risk before a breach occurs. Quell determines which zero-days are reachable in an organization's environment, whether existing controls would stop them, and which compensating control changes would disrupt the exploit immediately. This approach has proven effective, with organizations using Tuskira reducing breachable exposure by up to 99%.

The traditional approach to zero-day defense, which involves racing to patch vulnerabilities, is no longer effective in today's fast-paced threat landscape. With the rise of AI-powered exploit discovery, the window between disclosure and weaponization has shrunk to minutes, making it impossible to patch vulnerabilities quickly enough. Quell addresses this challenge by focusing on faster mitigation, disrupting exploitation through existing defenses, and identifying the most effective compensating control changes to close breachable paths.

Quell is built on Tuskira's Security Context Graph and digital twin, which correlates exposure, identity, network reachability, and existing controls to provide a comprehensive view of an organization's security posture. The solution extends to newly disclosed threats through Zero Day Response, ingesting live threat intelligence and mapping exploit preconditions against the environment to identify reachable, exploitable, and undefended assets. With Quell, enterprises can identify and neutralize zero-day exposure windows in hours, without waiting for a vendor patch, and track breach resilience gains over time.
