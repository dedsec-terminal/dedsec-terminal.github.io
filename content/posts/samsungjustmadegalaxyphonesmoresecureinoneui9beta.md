Samsung has enhanced the security of its Galaxy phones with the One UI 9 beta. A key feature is the integration of Lockdown mode into the power menu, which can be accessed through the power off, restart, and emergency options screen. When initiated, Lockdown mode disables biometric authentication, requiring a PIN or password to unlock, power off, or restart the device.

In the One UI 9 beta, activating Lockdown mode returns users to the lock screen, whereas in One UI 8.5, it would return them to the previous app. To access the power menu in One UI 8.5, users had to long-press the power button or the power and volume down buttons, and Lockdown mode was a separate option. The new implementation streamlines the process and adds an extra layer of security.

The One UI 9 beta also introduces other security and feature updates, including a unified call log and settings for managing Android 17's Audio Hardening feature. These changes aim to improve the overall security and user experience of Samsung's Galaxy phones and tablets. With the One UI 9 beta, Samsung is taking steps to enhance the protection of user data and provide a more secure mobile experience.
