Xint.io has introduced Xint Pulse, a black-box autonomous penetration testing tool designed for on-demand security assessments of web applications. This tool provides product security teams with timely, one-off penetration tests, making it accessible to organizations of all sizes. Xint Pulse utilizes a combination of Large Language Models (LLMs) and a proprietary orchestration engine to identify and validate critical security vulnerabilities in web application code.

Xint Pulse can analyze millions of lines of source code, configuration files, and binaries in under 12 hours, offering the same depth and detail as a human penetration tester. The tool is ideal for one-time tests, such as compliance submissions, vendor assessments, or pre-release validation, and is priced at $3,000 per application scan. This makes it an affordable option for small organizations that cannot afford a full, human-led pentest. With Xint Pulse, organizations can scan one application and retest up to three times within a 30-day period.

The tool currently focuses on black-box testing, where organizations provide the URL for their web-based applications, and Xint's autonomous AI agents probe for intrusion feasibility. The output is a compliance-ready report providing severity scores, trigger conditions, exploit impact, and suggested remediations. By the end of the year, Xint Pulse will also launch white-box testing, allowing product security teams to upload their source code and scan it line-by-line to find the root cause of vulnerabilities at the code level. This will further enhance the tool's capabilities and provide more comprehensive security assessments for organizations.
