Adobe has released patches for multiple high-severity security flaws in Adobe ColdFusion and Adobe Campaign Classic. The ColdFusion updates address critical and important vulnerabilities that could lead to arbitrary code execution, privilege escalation, and security feature bypass. A total of seven vulnerabilities were patched, including five with a CVSS score of 10.0, which is the highest severity rating.

The vulnerabilities in ColdFusion include unrestricted upload of file with dangerous type, improper input validation, and path traversal vulnerabilities, all of which could lead to arbitrary code execution. The issues have been addressed in ColdFusion 2023 Update 21 and ColdFusion 2025 Update 10. Security researchers Anirudh Anand, Matan Sandori, and 2Bsecure have been credited with discovering and reporting some of the vulnerabilities.

Adobe has also patched a critical flaw in Adobe Campaign Classic, which could result in arbitrary code execution. The vulnerability, tracked as CVE-2026-48286, has been patched in version ACC v7: 7.4.3 build 9397. Adobe noted that it has not found any exploits in the wild for any of the issues addressed and is moving to twice-monthly publication of security bulletins and advisories to keep up with the accelerated vulnerability discovery using artificial intelligence (AI) models.
