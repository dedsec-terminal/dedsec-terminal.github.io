Jetico has expanded its BestCrypt Data Shelter solution to include centrally managed enterprise data access control for sensitive files. This extension allows security teams to define and enforce policies governing which applications, processes, and users can access protected files, aligning with zero-trust security principles. The default-deny model restricts access by default, only allowing trusted applications and authorized users to interact with sensitive data.

The solution enables administrators to create, distribute, and update access control policies across endpoints from a centralized management console. Each policy specifies which trusted applications, processes, and users can interact with designated files, reinforcing existing encryption, authentication, and perimeter security. BestCrypt Data Shelter can be used to limit insider and process risk, reduce ransomware risk, and support immutable backup strategies. It also integrates with Jetico's AI-powered BCWipe Search capabilities for data discovery and classification.

The expanded BestCrypt Data Shelter solution is part of Jetico's endpoint data protection framework, which helps organizations discover, protect, and securely erase sensitive data. The company is inviting a limited group of organizations to evaluate the solution through its enterprise pilot program, focusing on proactive data access control against ransomware, unauthorized access, and data leakage. Jetico aims to provide organizations with a unified endpoint data protection strategy that maintains visibility, control, and compliance throughout the data lifecycle, supporting compliance and audit readiness under strict regulations.
