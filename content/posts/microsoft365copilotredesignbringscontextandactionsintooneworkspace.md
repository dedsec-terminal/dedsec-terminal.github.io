Microsoft 365 Copilot, an AI assistant, has undergone a redesign to provide a single, flexible entry point across Microsoft 365 apps. The new interface is built around user intent, using the design principle of progressive disclosure to reveal additional capabilities as needed. This allows users to start with a focused interface and access more features as they work.

The redesigned interface features a task-aware workspace where users can describe tasks and goals, and Copilot surfaces relevant tools and controls to assist. The experience is powered by Work IQ, an intelligence layer that personalizes Microsoft 365 Copilot and understands context, relationships, and work patterns. The app also loads twice as fast and includes performance and reliability improvements. Additionally, Copilot can be invoked directly within Microsoft 365 apps, such as Word, Excel, PowerPoint, and Outlook, allowing users to work within their current application while Copilot takes action.

The redesign has led to increased usage of Copilot, with a 27% increase in Word, 33% in Excel, 43% in PowerPoint, and 30% in Outlook. According to Jon Friedman, Chief Design Officer for Microsoft 365, the updates represent a broader shift in how AI is designed for work, moving from individual features to connected experiences and shaping technology around how people actually work. The goal is to create a more seamless and intuitive experience for users, allowing them to work more efficiently and effectively with the assistance of AI.
